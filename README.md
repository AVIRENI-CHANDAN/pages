## Add a new project reference

1. Copy the template:
   `projects/_template/index.html` -> `projects/<project-slug>/index.html`
2. Replace bracketed placeholders in the copied file.
3. Add a new project card in `index.html` under `Active Projects`.
4. Link the card to `/projects/<project-slug>/index.html`.
